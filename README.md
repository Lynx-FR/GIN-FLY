# GIN-FLY
GIN-FLY (Getting Info of Network Free, Light, and Yours) is an open source network analyser tool written in Python, you can look, change, and redistribute the code freely.

**DISCLAMER** 

Always use G-FLY on authorized networks or else you could face legal trouble and fines use it at your # OWN risk.

Note that the program is written for Linux but it should also works on macOS and BSDs  #HOWEVER G-FLY does #NOT works whit any version of Windows.
